# Menos es Más

Los iconos son vitales para la navegación visual. Elegí **Lucide React** porque:

1.  **Tree Shakable**: Solo importo el icono que uso. Si uso `<Menu />`, solo ese SVG entra en el bundle final.
2.  **Stroke Ajustable**: Puedo controlar el grosor de línea para que coincida con mi tipografía.
3.  **Consistencia**: Todos los iconos siguen la misma grid y estilo visual.

En el formulario de contacto, uso iconos para feedback visual (un check verde para éxito, una alerta roja para error). Esto mejora la accesibilidad cognitiva, ayudando a los usuarios a entender el estado de la aplicación instantáneamente sin leer texto.
