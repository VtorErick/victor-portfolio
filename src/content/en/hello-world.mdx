# Hola Mundo

Bienvenido a mi nuevo portafolio. He decidido integrar este blog directamente en mi sitio personal para tener control total sobre mi contenido.

## ¿Por qué MDX?

MDX me permite escribir contenido en Markdown pero incrustar componentes de React directamente en el flujo. Es perfecto para mostrar:
- Snippets de código interactivos
- Componentes visuales personalizados
- Gráficas y datos

```javascript
const Greeting = () => {
  return <h1>Hola desde el código!</h1>
}
```

Espero que disfrutes leyendo mis artículos tanto como yo disfruto escribiéndolos.
